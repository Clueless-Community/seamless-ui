## General convention to follow :

### **Border-radius** :

Border radius of each card is `rounded-2xl` 

### **Border :**

- border thickness : `border`
- border color : `none`

### **Paddings :**

The padding could be variable depending on the card sizes and UI requirements but normally it is `p-6`

### Font :

|  | heading | Normal text | sub text | badge type |
| --- | --- | --- | --- | --- |
| size | text-2xl | text-lg | text-sm | text-sm / text-md |
| color | text-gray-800 | text-gray-800 / text-gray-600 ( according to the design) | text-gray-600 | text-gray-400 |
| font | semi-bold | semibold / normal | normal | normal |

### Width & heights :

The width of the component totally depends on the design.

But we usually use these 3 widths üîΩ

- `w-48` ‚Üí for small components
- `w-72` ‚Üí for medium components
- `w-96` ‚Üí for big components

The height of the components depends on the amount of content on it. We need to make it fluid. **Giving a fixed height is not recommended at all.** 

---

### Anchor tag :

- **color** : `text-blue-600`
- **underline** : On hover there is an underline ‚Üí `hover:underline`
- **underline offset** : `underline-offset-4`
