The badge component can be used to complement other elements such as buttons or text elements as a label or to show the count of a given data.

Such as the number of comments for an article or how much time has passed by since a comment has been made.

![image](https://user-images.githubusercontent.com/83635276/205807536-98d7ddef-179e-4c91-9ea4-cc053b86d305.png)

## Types of Badges ðŸ”½

1. Default badge
2. Badges with icon
3. Notification badge
4. Button with badge
5. Badge with icon only

---

## General must known design system guideline :

### Font

For all buttons font is : **`font-medium`**

### Border Radius ðŸ”²

The normal border-radius for the components are `rounded`.

### Hover Effect :

For the hover effect on the button it is just one shade darker than the original button. Use It wherever neccessary.

## Icon guidelines:

Mostly all the icons are taken from â†’ [Hero Icons](https://heroicons.com/)

`Icon-text-gap` :  We follow a convention of equal margin between the text and the icon as padding in y  axis `py`. and always the margin is given from the side of icon.

|  | Width | Type |
| --- | --- | --- |
| Small | w-4 & h-4 | Hero Icons mini |
| Medium | w-5 & h-5 | Hero Icons mini |
| Large | w-6 & h-6 | Hero Icons filled |
