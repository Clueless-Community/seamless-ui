# Form group Design System ‚≠ê

Form Group can be wrapped around components used in a Form, such as Input or Radio. Form Group has three sizes ‚Äî large, medium and small. The size controls the spacing between the groups. Form Text should be used to display informational text that helps the user with data entry.

![Screenshot 2022-11-10 at 12 23 10 AM](https://i.postimg.cc/NMLHbs5c/Screenshot-2022-12-31-at-4-28-10-PM.png)

## General convention to follow :


### Color üé®

|  | Background | Border | Active | Inactive |
| --- | --- | --- | --- | --- |
| Color Code in Tailwind | bg-gray-100 | border-gray-200 | bg-green-400 | bg-red-400 | bg slate-900 | bg-gray-400 |
 


 
|  | Background | Background |

| --- | --- | --- | 
| Color Code in Tailwind | bg-gray-100 |bg-white| bg-gray-300 | 
 

Color of `text` will depend on the `bg` color
| Background | Dark | Light |
| --- | --- | --- |
| Text | Light | Dark |
| Code in Tailwind | text-white | text-gray-200 | text-slate-600| text-black | text-gray-800 | text-gray-500 |

`Note` : Color of the text `[color]` will depend on the color of the background

### **Border-radius** :

| small | medium | large |
| --- | --- | --- |
| rounded-md | rounded-lg | rounded-xl |

### **Border :**

- border thickness : `border-2`
- border color : `border-gray-200`
- border focus color : `focus:border-blue-600`


### **Paddings :**

| small | medium | large |
| --- | --- | --- |
| p-2 | p-3 | p-5 |

### Font :

| small | medium | large |
| --- | --- | --- |
| text-sm | text-md | text-lg | 

### Width :

With is always set to full `w-full` so that the user can use the external div to control the width of the input box easily.

---

### Disabled State :

In the disabled state we use a background color with `bg-gray-300`

### Buttons :

We need to use the already built buttons for this. **[Buttons](https://github.com/Clueless-Community/seamless-ui/tree/main/Buttons)** üîó¬†. Don‚Äôt use any buttons outside this.

### Icons :

Icons must be used from **[Heroicons](https://heroicons.com/)** üñáÔ∏è

**Note :** You need to use a solid 24X24 SVG form heroicons.
