
# ðŸ“’ Developer style guide for Dropdown

The dropdown component can be used to show a list of menu items when clicking on an element such as a button and hiding it when focusing outside of the triggering element.

![image](https://i.postimg.cc/T1LGqnNV/Screenshot-2022-12-31-at-1-08-49-AM.png)

---

## General must known design system guideline :

### Color ðŸŽ¨

| Color | Yellow | Gray | Dark-Gray |
| --- | --- | --- | --- |
| Code in Tailwind | bg-blue-700 | bg-blue-600 | bg-gray-200 |

Color of `text` will depend on the `bg` color
| Background | Dark | Light |
| --- | --- | --- |
| Text | Light | Dark |
| Code in Tailwind | text-gray-400 | text-white | text-gray-600 |

`Note` : Color of the text `[color]` will depend on the color of the background

### Font

For all Dropdown font is : **`font-medium`**

---

## Icon guidelines:

Mostly all the icons are taken from â†’ [Hero Icons](https://heroicons.com/)

`Icon-text-gap` :  We follow a convention of equal margin between the text and the icon as padding in y  axis `py`. and always the margin is given from the side of icon.

|  | Width | Type |
| --- | --- | --- |
| Small | w-4 & h-4 | Hero Icons mini |
| Medium | w-5 & h-5 | Hero Icons mini |
| Large | w-6 & h-6 | Hero Icons filled |
