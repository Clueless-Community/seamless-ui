## General convention to follow :

### **Color**
- For Dark : `bg-slate-900`
- For Blue : `blue-600`
- For White : `bg-white` 

### **Border-radius** : 
Border radius of each card is `rounded-2xl`

### **Paddings :**

The padding could be variable depending on the card sizes and UI requirements but normally it is `px-9` `py-10`

### Font :

|  | heading | sub text | 
| --- | --- | --- |
| size | text-2xl | text-sm |
| color | text-white / text-blue-600 / text-black | text-gray-600 / text-gray-400 |
| font | font-bold | font-normal |
| line height | leading-8 | leading-4 |

### Width & heights :

The width of the component totally depends on the design.

But we usually use these 3 widths üîΩ

- `w-48` ‚Üí for small components
- `w-72` ‚Üí for medium components
- `w-96` ‚Üí for big components

The height of the components depends on the amount of content on it. We need to make it fluid. **Giving a fixed height is not recommended at all.** 

## Buttons :

We need to use the already built buttons for this. **[Buttons](https://github.com/Clueless-Community/seamless-ui/tree/main/Buttons)** üîó¬†. Don‚Äôt use any buttons outside this.
